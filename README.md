# Simple Tweeter

A simple tweeter client for purpose of learning Objective-C and Cocoas Touch API. It also leverages some excellent open sources:

* [AFNetworking](https://github.com/AFNetworking/AFNetworking)
* [RestKit](https://github.com/RestKit/RestKit)
* [TDOauth](https://github.com/tweetdeck/TDOAuth)
